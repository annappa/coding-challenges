Given 2 arrays, create a function that let's a user know (true/false) whether these two arrays contain any common items

For Example:

```text
array1 = ['a', 'b', 'c', 'x'];

array2 = ['z', 'y', 'i'];

should return false.
```
-----------
```text
array1 = ['a', 'b', 'c', 'x'];

array2 = ['z', 'y', 'x']

should return true.
```

2 parameters - arrays - no size limit
